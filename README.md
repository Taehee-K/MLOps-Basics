## Usage

### Installation

```
git clone https://github.com/Taehee-K/MLOps-Basics.git
cd MLOps-Basics
pip install -r requirements.txt
```

### Train

```
python train.py
```

### Monitoring

Visualize TensorBoard Logs

```
tensorboard --logdir logs/cola
```

### Inference

```
python inference.py
```
